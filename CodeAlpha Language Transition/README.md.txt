# 🌍 AI Language Translation Tool

## 📌 Project Overview

The AI Language Translation Tool is a web application developed using **Python** and **Streamlit** that allows users to translate text between multiple languages using Google Translate.

The application provides a simple and interactive interface where users can enter text, choose the source and target languages, and instantly receive the translated output.

---

## 🚀 Features

- 🌍 Supports multiple languages
- 🔄 Translate text instantly
- 📝 User-friendly interface
- ⚡ Fast and accurate translation
- 📋 Copy translated text
- 📱 Responsive Streamlit UI

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Deep Translator
- Google Translate API
